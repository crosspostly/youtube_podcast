# Constitution

**Незыблемые правила проекта:**

- Архитектура — модульный подход, Sliced Design
- Запрещены `any`, `as any`, и неявные типы в TypeScript
- Основа: только OSS лицензии для внешних пакетов
- В клиентском коде не должно быть авторизации или персональных данных
- Использование Gemini/OpenAI API только через абстракции
- Отказоустойчивость API (должен быть fallback на резервные сервисы/ключи)
- Чистый публичный repo code review — ни один PR не мержится без ссылки на задачу и чек-лист
- Никаких автотестов на MVP — только ручная валидация по чек-листу
- Все взаимодействие — исключительно через UI (без внешних тулзов для студента/пользователя)

# Specify: Человеческое описание проекта

Приложение "Студия Подкастов с ИИ" — это клиентская платформа, автоматизирующая создание аудиовизуальных нарративов.
- Генерация многоглавных сценариев по пользовательской теме или загруженной базе знаний.
- Голосовая озвучка (мультиспикер) с помощью Google Gemini TTS, поддержка нескольких языков.
- Автоматический подбор музыки, SFX и создание микса с интеллектуальным сведением на клиенте.
- Генерация уникальных фонов и YouTube обложек через AI, визуальный редактор для пост-обработки.
- Сборка финального видео (Ken Burns effect, субтитры).
- Быстрое редактирование, возможность вносить правки в любой элемент (глава, аудио, изображение).
- Вся история — в LocalStorage браузера, работа — без регистрации, приватно.
- Использование нескольких open-source/бесплатных AI API для “fallback” в случае перегрузки одного из сервисов.
- Без серверной логики и авторизации.

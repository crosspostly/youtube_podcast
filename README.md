## 1. Видение проекта: ИИ как творческий партнер

### Идея
**Студия Подкастов с ИИ** — это не просто генератор контента, а полноценная творческая среда, созданная для расширения возможностей авторов, маркетологов и всех, кто хочет рассказывать истории. Мы стремимся демократизировать создание высококачественных аудио- и видео-нарративов, делая процесс быстрым, интуитивным и вдохновляющим.

### Цель
Наша цель — автоматизировать рутинные и технически сложные аспекты производства (написание сценария, озвучка, саунд-дизайн, создание визуальных материалов), освобождая время автора для самого важного — творчества и режиссуры. Приложение берет на себя роль мультидисциплинарной команды, позволяя одному человеку достигать результатов, на которые раньше требовались недели работы и несколько специалистов.

### Философия
Мы верим в **партнерство человека и ИИ**. Это не "волшебная кнопка", которая делает все за вас. Это итеративный инструмент. ИИ создает мощную, профессиональную основу, а вы выступаете в роли **режиссера**, который доводит ее до совершенства: меняет реплики, перегенерирует изображения, подбирает идеальный саундтрек и настраивает финальный микс.

## 2. Ключевые возможности: Ваша автоматизированная команда

### Batch Export

После завершения массовой генерации проектов, используйте кнопку **"Скачать все проекты"** для автоматической упаковки всех завершённых проектов в единый архив.

**Структура batch-архива:**
```
batch_projects_2025-11-23.zip
├── project_1_название/
│   ├── chapters/
│   │   ├── chapter_01/
│   │   └── chapter_02/
│   ├── assemble_video.bat
│   └── project_metadata.json
├── project_2_название/
└── project_3_название/
```

Каждый проект упакован по [chapter-based архитектуре](./CHAPTER_PACKAGING.md).

### Подробное логирование

Все этапы упаковки логируются в журнал запросов (кнопка "Журнал" в правом верхнем углу):
- Начало/завершение упаковки каждого проекта
- Обработка каждой главы с указанием ресурсов
- Прогресс общего архива
- Детали ошибок при возникновении

(Остальные разделы без изменений)

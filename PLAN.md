# PLAN: Техническое описание и стек

- **Frontend:** React 18, TypeScript 5, TailwindCSS
- **AI:** Google Gemini API (2.5-flash), OpenRouter API (fallback)
- **TTS/ASR/SFX:** Google Gemini TTS, Freesound.org API, Jamendo API (music)
- **Video:** ffmpeg.wasm клиентская сборка (mp4, webm)
- **Данные:** IndexedDB (кэш голосов), LocalStorage (история проектов, состояние генерации)
- **Нет бэкенда**, нет регистрации (статическая сборка)
- **Абстракции API:** Все обращения к сторонним сервисам через единый слой (services), с падением на резерв при любой ошибке.
- **Обработка ошибок:** Критически важные операции (создание видео, генерация глав, TTS) требуют информирования пользователя и предложения действий
- **Модули:**
  - Генерация сценария/глав
  - Мульти-TTS озвучка и SFX поиск/установка
  - Генерация фонов/обложек
  - Видеосборка, наложение субтитров
  - Панели тестирования (AI, видео, музыка отдельно)
  - Система хранения истории проектов (CRUD)
  - UI-менеджмент, уведомления и ошибки

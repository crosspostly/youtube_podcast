# Pre-Publication Checklist (50+ Optimization)

Use this checklist before publishing **every** YouTube podcast. Print it or duplicate per episode to log approvals.

## Content Structure
- [ ] Retention anchor every 8–10 minutes? (Count: ____)
- [ ] Micro-conclusion every ~10 minutes? (Count: ____)
- [ ] Chapter markers listed in description? (00:00, 08:30, ...)
- [ ] Open loop for next video in final segment?
- [ ] Total duration 40–50 minutes? (Actual: ____ min)

## Script Quality
- [ ] Each chapter ≥ 8,500 characters of dialogue?
- [ ] Characters match Wisdom Exchange archetypes?
- [ ] Signposting phrases near 12, 25, 38 minutes?
- [ ] Pauses ≥0.8s between sentences and ≥1.5s between paragraphs?
- [ ] No slang/memes/Gen-Z jargon?

## Audio Quality
- [ ] Speech clarity boost (2–4 kHz +3 dB) confirmed in FFmpeg logs?
- [ ] Background music volume ≤15% (metadata + mix)?
- [ ] Atmospheric SFX ≤20% volume?
- [ ] Sudden SFX ≤40% volume?
- [ ] Compression active (3:1 ratio @ -18 dB)?

## Visual Accessibility
- [ ] Subtitle font = Arial Bold, 32 px?
- [ ] Outline = 3 px black + high-contrast fill?
- [ ] MarginV = 60 (safe zone)?
- [ ] Each image displayed ≥5 seconds?
- [ ] Visuals readable from 3 meters on a TV?

## Titles & Thumbnails
- [ ] Titles use authority/credibility framing (no ALL CAPS)?
- [ ] Generated ≥3 title options and selected best?
- [ ] Thumbnail font = serif/high-legibility, ≤4 words, no clickbait?
- [ ] Palette = sophisticated (navy/gold, charcoal/ivory, deep green/copper)?

## Metadata & Description
- [ ] Description professional, structured, no hype language?
- [ ] Chapter timestamps included?
- [ ] Tags age-appropriate (no “shocking”, “crazy”)?

## Final Confidence Gate
- [ ] Would a 60-year-old find this credible and respectful? (Yes / No)
- [ ] Respects viewer intelligence? (Yes / No)
- [ ] Audio clear on TV speakers + headphones? (Yes / No)
- [ ] Visuals comfortable for long-form viewing? (Yes / No)
